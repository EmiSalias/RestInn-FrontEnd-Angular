Sobre RestInn

Conocé más sobre el proyecto y el equipo detrás de este sistema integral de gestión hotelera.

El Proyecto

RestInn es una plataforma de gestión hotelera integral desarrollada como nuestro Trabajo Práctico Final. Utiliza una arquitectura moderna basada en Angular para el frontend y Spring Boot para la API RESTful del backend.

Misión: Proporcionar una solución tecnológica robusta, centralizada y fácil de usar para la gestión hotelera, permitiendo administrar eficientemente reservas, habitaciones, usuarios y facturación.

Alcance: El sistema es independiente, aunque emplea bibliotecas y herramientas de terceros para funciones clave como seguridad, generación de facturas en PDF y gestión de notificaciones por correo electrónico.

El objetivo final es optimizar los procesos administrativos y mejorar la experiencia tanto de los huéspedes como del personal del hotel.

El Equipo
